## Cryptography

| Algorithm | Purpose |
| --------- | ------- |
| Ed25519 | Signatures on messages, blocks, certificates |
| AES-256-GCM | Encrypted credential storage |
| X25519 | Key exchange for secure key transfer |
| SHA-256 | Block chaining, content hashing |

All cryptographic operations use well-audited libraries (libsodium via Node.js crypto).

## Input Protection

The InputSanitizer detects and blocks:

| Pattern | Description |
| ------- | ----------- |
| SQL injection | `'; DROP TABLE`, `UNION SELECT` |
| XSS | `<script>`, `javascript:` |
| Command injection | `; rm -rf`, `$()`, backticks |
| Path traversal | `../`, `/etc/passwd` |
| SSRF | Internal IP addresses, localhost |
| Prompt injection | Instruction override attempts |
| Base64 encoded attacks | Decoded and re-scanned |
| Zero-width characters | Stripped to prevent obfuscation |

Rate limiting prevents abuse:
- Per-user request limits
- Per-IP limits
- Automatic backoff for violations

## Output Protection

The OutputGate monitors 9 sensitive data types:

| Type | Default Action |
| ---- | -------------- |
| API keys | Block |
| Passwords | Block |
| Private keys | Block |
| SSN/Tax IDs | Redact |
| Credit cards | Redact |
| Phone numbers | Warn |
| Email addresses | Warn |
| Physical addresses | Warn |
| Medical info | Redact |

Actions per type:
- **Block**: Response rejected, not sent
- **Redact**: Content replaced with `[REDACTED]`
- **Warn**: Response sent but flagged for review

## Trust Management

See [Trust Levels](/networking/trust-levels) for the full trust model.

Key features:
- 5 tiers from Blocked (-1) to Owner (0)
- Action-based permissions
- Quarantine for low-trust content
- Memory guard integration

## Memory Protection

The Memory Guard enforces:

| Protection | Description |
| ---------- | ----------- |
| Write protection | Certain memory types require elevated trust |
| Read protection | Sensitive memories hidden from low-trust entities |
| Provenance verification | Memories tracked to their source |
| Federation redaction | Automatic redaction before sharing |

## Audit Trail

All operations are logged:

```typescript
interface AuditEntry {
  timestamp: string;
  actor: string;
  action: string;
  resource: string;
  result: "success" | "blocked" | "error";
  details: Record<string, unknown>;
  trustLevel: number;
}
```

Storage:
- Local SQLite for immediate access
- Optional export to external SIEM
- Configurable retention period
