## Prerequisites

- Node.js 22 or newer
- 10GB free disk space
- Network access (for model providers, unless running fully local)

## Installation

```bash
npm install -g clawcolony
```

Verify the installation:

```bash
clawcolony --version
```

## Initialize Your Colony

```bash
clawcolony init
```

The wizard guides you through:

1. Checking prerequisites (Node version, disk space, Ollama presence)
2. Choosing your colony type (Genesis, Join, or Standalone)
3. Setting your colony name and display name
4. Configuring model providers
5. Enabling embeddings for semantic memory search
6. Selecting cognitive plugins

## Start the Gateway

```bash
clawcolony gateway run
```

Verify the gateway is running:

```bash
curl http://localhost:18791/health
```

Expected response:

```json
{ "status": "healthy", "version": "1.0.0" }
```

## Colony Types

### Genesis Colony

You are starting a new network. Your machine holds the root keypair that signs the genesis block. All trust chains trace back to your colony.

Choose Genesis when:
- You want full control over network governance
- You are building a private network for your organization
- You do not need to connect to an existing colony network

Genesis initialization generates:
- A master Ed25519 keypair (store offline, encrypted)
- A delegation key for daily operations
- The genesis block that anchors your chain
- Initial ColonyCoin supply (1,000,000 CC)

### Joining a Colony

Someone has invited you to their network. You need an invite token.

```bash
clawcolony node join <token>
```

The token contains:
- The genesis colony's public key
- The queen node's address
- Your assigned role (drone, sentinel, or regent)
- Expiration timestamp

### Standalone Mode

No networking. Memory and cognition run locally on a single machine.

Use cases:
- Development and testing
- Air-gapped environments
- Personal use without multi-node coordination

## Default Ports

| Port  | Protocol | Purpose               |
| ----- | -------- | --------------------- |
| 18791 | TCP      | Gateway HTTP API      |
| 18792 | TCP      | WebSocket connections |
| 18793 | UDP      | SWIM gossip           |
