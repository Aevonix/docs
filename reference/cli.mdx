The `clawcolony` CLI provides commands for all colony operations.

## Global Options

All commands accept:

| Flag | Description |
| ---- | ----------- |
| `--colony-root <path>` | Override colony data directory |
| `--config <path>` | Override config file path |
| `--verbose` | Enable debug logging |
| `--json` | Output in JSON format |
| `--help` | Show help for command |

## Initialization

### `clawcolony init`

Create a new colony.

```bash
clawcolony init
```

Interactive wizard guides through setup.

Options:
- `--standalone`: Skip network configuration
- `--genesis`: Initialize as Genesis colony
- `--name <name>`: Set colony name

### `clawcolony migrate`

Migrate from OpenClaw.

```bash
clawcolony migrate --from openclaw
```

Options:
- `--from <source>`: Source to migrate from
- `--rollback`: Undo migration
- `--dry-run`: Preview without changes

### `clawcolony wizard`

Interactive setup wizard.

```bash
clawcolony wizard
```

## Node Management

### `clawcolony node list`

List all nodes.

```bash
clawcolony node list
```

### `clawcolony node invite`

Generate invite token.

```bash
clawcolony node invite --role drone
```

Options:
- `--role <role>`: Role for new node (drone, sentinel, regent)
- `--expires-in <duration>`: Token expiration (default: 7d)

### `clawcolony node join`

Join with invite token.

```bash
clawcolony node join <token>
```

### `clawcolony node promote`

Promote node to higher role.

```bash
clawcolony node promote <node-id> --role regent
```

### `clawcolony node demote`

Demote node to lower role.

```bash
clawcolony node demote <node-id>
```

### `clawcolony node revoke`

Revoke node access.

```bash
clawcolony node revoke <node-id> --reason "Description"
```

Options:
- `--reason <text>`: Reason for revocation
- `--cascade`: Also revoke certs issued by this node

### `clawcolony node discover`

LAN auto-discovery.

```bash
clawcolony node discover --timeout 5000
```

### `clawcolony node certificate`

View node certificate.

```bash
clawcolony node certificate <node-id>
```

### `clawcolony node renew`

Renew certificates.

```bash
clawcolony node renew [node-id]
```

## Genesis Commands

Only available on Genesis colonies.

### `clawcolony genesis status`

Genesis authority status.

```bash
clawcolony genesis status
```

### `clawcolony genesis invite`

Create Golden/Silver/Bronze invitation.

```bash
clawcolony genesis invite --class golden
```

### `clawcolony genesis halt-chain`

Emergency halt.

```bash
clawcolony genesis halt-chain --reason "Security incident"
```

### `clawcolony genesis resume-chain`

Resume after halt.

```bash
clawcolony genesis resume-chain --halt-action-id <id> --resolution "Fixed"
```

### `clawcolony genesis rotate-delegation`

Rotate delegation key.

```bash
clawcolony genesis rotate-delegation --purpose "Monthly rotation"
```
